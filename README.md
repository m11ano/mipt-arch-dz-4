# МФТИ 2025, Архитектура ПО, ДЗ №4

## Рефакторинг приложения

-   Оставил в коде комментарии в основных изменения
-   Сделал фабрику для инициализации классов-конвертеров
-   Вынес работу с внешним API в отдельный сервис.
-   Стратегия инициализации этого сервиса — **Multiton** (разновидность Singleton).
-   В абстрактном классе оставлен один ключевой метод `convert`, в конкретных конвертерах теперь содержится только бизнес-логика.
-   Вынес конструктор логгера в отдельный класс.
-   Доделал асинхронность: методы конвертации запускаются параллельно.  
    На уровне сервиса при обращении к внешнему API используется блокировка для защиты от множественных одновременных запросов.
